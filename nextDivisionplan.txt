- Only organizers authenticate to create/manage events.

Shared Rules
------------
1) Do not edit each other's owned files without discussion.
2) Open small PRs daily.
3) Rebase with main before PR.
4) Keep API response format consistent:
   { success: true/false, message: string, data: object|null }
5) If request/response shape changes, update docs first.
